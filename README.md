# 📘 Facebook Clone Website

A simple **Facebook Clone Website** developed as a practice project while learning **Git** and **GitHub**. This project was created to improve my understanding of version control, repository management, and front-end web development. It replicates the basic layout and user interface of Facebook using modern web technologies.

Although this project does not include backend functionality such as user authentication, databases, or real-time messaging, it demonstrates how to build a responsive and visually appealing social media interface.

## 🚀 Features

* Responsive Facebook-inspired user interface
* Navigation bar
* Sidebar menu
* News feed layout
* Profile section
* Stories section
* Posts and feed design
* Clean and modern UI
* Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## 🎯 Purpose of the Project

The primary objective of this project was to:

* Learn Git and GitHub workflows
* Practice version control using Git
* Understand repository creation and management
* Improve HTML, CSS, and JavaScript skills
* Gain hands-on experience building a real-world UI clone
